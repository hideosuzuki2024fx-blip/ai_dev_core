# TrendHook — LP Draft

## ⚡ Hero
“いま”刺さる投稿を、時刻×地域×話題クラスタから自動生成。  
SNS投稿の“旬”を逃さない。  

**CTA:** 無料で先行アクセス登録  

---

## 🚨 Problem
- 何を・いつ・どこで投稿すべきか分からない  
- ハッシュタグ選定や文案に時間がかかる  

---

## 💡 Solution
- 時刻×地域×話題クラスタのTop5を自動提示  
- 投稿文・タグ・時刻を1クリックで提案  

---

## 🧩 Core Features
- 話題クラスタ抽出  
- 投稿文3案（短/標準/長）  
- ハッシュタグ自動提案（競合度・関連度）  
- 投稿スケジュール提案（現地時刻基準）
