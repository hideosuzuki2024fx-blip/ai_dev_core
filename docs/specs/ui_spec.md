# UI Specification
Last Updated: 2025-11-13 11:08:15

## Streamlit UI Structure
- File selector
- Thumbnail grid (drag & drop)
- Feature analysis trigger
- Preview window
- Progress bar
- Final render button

## UI Component Responsibilities

### file_selector.py
- Accept video files
- Validate extensions

### thumbnail_grid.py
- Display video thumbnails
- Support drag & drop ordering

### preview_window.py
- Show transition preview

### progress_display.py
- Display %
