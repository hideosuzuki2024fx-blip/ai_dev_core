## ⚙️ 開発方針
- 中略・省略禁止：すべてのコードはコピペで動作する完全形を提示
- Semantic Diff運用：無意味な差分を作らない
- AI⇄人間分担：AIが自動生成、人間は承認・公開操作のみ
- 再現性重視：PowerShell／Python単体で即再構築可能
